# 開發時間報表

## 功能

顯示每日開發時間統計，支援按日期彙總和按 WBS 任務分類。

## 資料來源

**必須** 從 `.claude/taskmaster-data/timelog.jsonl` 讀取時間記錄。

每筆記錄格式：
```json
{"session_id":"...","date":"2026-03-24","start":"14:07","duration_ms":3600000,"cost_usd":12.50,"task":"2.1"}
```

如果檔案不存在或無資料，提示：「尚無時間記錄。開始開發後會自動追蹤。」

同時讀取 `.claude/taskmaster-data/.session-snapshot` 取得**當前 session** 的即時時間（尚未歸檔的）。

## 流程

1. 讀取 `timelog.jsonl`，每行一筆 JSON
2. 讀取 `.session-snapshot`（如果存在），加入當前 session 的即時數據
3. 按日期分組彙總 `duration_ms` 和 `cost_usd`
4. 如有 WBS 檔案（`wbs.md`），交叉比對任務名稱和預估時間
5. 輸出報表

## 輸出格式

### 預設模式（每日彙總）

```
📊 開發時間報表
═══════════════════════════════════════════════
日期         Sessions  開發時間    花費
───────────────────────────────────────────────
2026-03-24   3         4h 35m     $42.84    ← 今天
2026-03-23   2         2h 10m     $18.50
2026-03-22   1         1h 20m     $11.00
───────────────────────────────────────────────
本週合計     6         8h 05m     $72.34
累計合計     18       24h 30m    $215.00
```

### `--by-task` 模式（按 WBS 任務彙總）

```
📊 任務時間報表
═══════════════════════════════════════════════════════════════
任務        名稱              預估     實際      差異     狀態
───────────────────────────────────────────────────────────────
1.1        專案初始化          0.5h    0h 25m    -5m     ✅
1.2        需求分析            2h      1h 50m    -10m    ✅
2.1        架構設計            2h      3h 15m    +1h15m  🔄
2.2        API 設計            1h      --        --      ⏳
3.1        核心功能開發         4h      --        --      ⏳
(未分類)    --                 --      1h 30m    --      --
───────────────────────────────────────────────────────────────
合計                          9.5h    7h 00m
```

「未分類」代表該 session 未指定任務的時間。

### `--today` 模式（今日詳細）

```
📊 今日開發時間
═══════════════════════════════════════════════
Session              開始    時長       任務     花費
───────────────────────────────────────────────
#1                   09:15   1h 20m    2.1      $8.50
#2                   14:07   2h 05m    2.1      $15.30
#3 (進行中)          19:30   0h 45m    2.2      $5.20
───────────────────────────────────────────────
今日合計                     4h 10m             $29.00
```

## 時間格式化規則

- `duration_ms` 轉換為 `Xh Ym` 格式
- 不足 1 分鐘顯示為 `<1m`
- 超過 8 小時的單日標記 ⚠️（提醒休息）

## 使用方式

```
/time-log              # 每日彙總（預設最近 7 天）
/time-log --by-task    # 按 WBS 任務彙總
/time-log --today      # 今日詳細
/time-log --all        # 所有記錄
/time-log --week       # 本週
/time-log --month      # 本月
```
