Phase 1：把散亂的「好煩」變成武器

你的原料：社群留言、客戶差評、朋友的抱怨、你自己半夜覺得「為什麼沒人做」的念頭。

全部都是散亂的。Phase 1 做一件事：把散亂，變結構。

「當 [誰] 在 [什麼時候] 想要 [什麼]，
但現在只能 [怎麼做]，讓他覺得 [多痛]。」

範例： 「當 Shopee 手作賣家在晚上回客服時想要快速回覆，但現在只能逐則手打 LINE，讓她覺得浪費時間又容易漏單。」

物理量畫布

四個格子。四個問題。全部要你自己填。AI 填不了。

#1 具體的人
我能說出 3 個人名嗎？
能打電話給他們嗎？
#2 正在付錢的痛
他們現在花多少錢/時間
在解決這個問題？
#3 手工可交付
不寫程式，
今晚能幫一個人做一次嗎？
#4 收款管道
我有現成的方式
收到錢嗎？

為什麼叫「物理量」？因為這四個東西可以測量。不是意見。不是感覺。是事實。

怎麼選題？ 不是選「最好的」。是選你認識最多人的那個。

人最多 = 驗證最快 = 犯錯成本最低。

找到位置

Phase 3：找到他們不屑做的事

大公司不是打不贏你。是打贏你對他們沒好處。

  • 你的市場太小——他們的業務團隊不屑。
  • 你的價格太低——他們跟進會拉低利潤。
  • 你砍掉的功能——他們的企業客戶不允許砍。

這就是你的機會。

兩種模式。只有兩種。

低端破壞

「你的工具太貴太複雜。我只需要其中 10%。」

目標：被過度服務的人。

新市場破壞

「我想解決，但根本沒工具。」

目標：非消費者。

怎麼判斷？看替代方案。

花大錢用複雜工具 低端破壞
用 Excel + 手動 + 忍耐 低端破壞
完全沒有解法 新市場破壞
免費工具但痛苦地用 新市場破壞

找到位置

Phase 4：不做清單比功能清單更重要

Strategy Canvas。橫軸是客戶在意的維度。縱軸是高低。

如果你的曲線跟競品長一樣，只是全部更低——你不是創新者。你只是更爛的版本。

你要的是：形狀不同。

範例

維度客服 SaaS手動 LINE你
價格 高 低 低
上手速度 低 高 高
功能數量 50+ 1 3
常見問答 有 無 有
訂單追蹤 有 半套 不做
多平台 有 無 不做

功能數量：3。不是 50。訂單追蹤：不做。多平台：不做。

因為月營收 3 萬的小賣家用 Excel 管訂單就夠了。他們需要的只是不要再重複回答尺寸和運費。

你的不做清單，必須比你的要做清單長。
如果不做比要做少——你砍得不夠。

不做清單：

  1. 不做完整線上課 — 學員只需要「這個動作對不對」的答案
  2. 不做社群 — 學員之間不需要互動，他們需要教練的眼睛
  3. 不做排課系統 — 不需要排課，學員隨時傳影片
  4. 不做 App — LINE 就夠了

差異化摘要：「我用『24 小時影片批改回覆』服務沒有任何付費管道的健身自主訓練者。刻意不做線上課、不做社群、不做 App。因為學員要的不是課，是『教練你看一下這樣對嗎』。」

紅隊打臉實錄：AI 怎麼挑戰小美的策略 紅隊

紅隊 Prompt 4B 的 AI 回覆摘錄：

挑戰 #2 — 「更便宜」不等於「破壞」

你的曲線在「價格」和「功能數」都比客服 SaaS 低，但在「常見問答」和「上手速度」比他們高。這形狀是對的——你不只是便宜貨，你在做不同的事。

但注意：「中文在地化」這個維度你跟手動 LINE 一樣高，這不是你的獨特優勢，是基本盤。考慮換成「設定代勞」——競品要客戶自己設定，你幫他做好，這才是差異。

挑戰 #5 — 「不做」的代價

你不做「訂單追蹤」。但如果小賣家的客戶問「我的東西寄了嗎？」——這個問題會佔常見問答的多少比例？如果超過 30%，你的自動回覆會有三成問題回答不了。驗證方式：問小美她每天被問的問題中，有幾個跟訂單有關。

小美的調整：她問了小美，訂單相關只佔 15%。安全。但她把「中文在地化」換成了「設定代勞」——這個維度競品完全沒有。

Phase 6：確認一件事——大公司不會來。

很多小產品不是做得不好。是大公司花一個週末就複製了。

所以你要問自己四個問題。

Q1：他們的錢從哪來？

你的客群是不是競品的利潤來源？如果不是——他們不會為了你動。 ✓

Q2：他們敢推低價版嗎？

如果推了，高價客戶會降級？如果會——他們不敢推。 ✓

Q3：他們砍得掉功能嗎？

企業客戶允許他們砍嗎？如果不允許——他們砍不了。 ✓

Q4：他們的業務在乎你嗎？

業務一單做 NT$50,000。你一單 NT$800。業務看你一眼嗎？不會。 ✓

判斷

4 個 ✓ 出手。他們不會來。
3 個 ✓ 出手。但要快。
2 個 ✓ 考慮調整。
≤ 1 個 回去重選。

→ 安全檢查通過。最後一步——把所有東西收成一張紙，然後按發送。

用 AI 跑這套框架，三件事你必須知道。

第一件：{{}} 是你填的。不是 AI。

Prompt 裡的雙大括號——那是你的真實資料。你的人名、你的觀察、你的數字。
如果你讓 AI 連這些都幫你編——你不是在用工具。你是在自我欺騙。

第二件：每個 Phase 都有一條 AI 過不去的線。

Phase 1 → 「我認識這個人嗎？」     ← 你答
Phase 2 → 物理量畫布               ← 你填
Phase 3 → 選哪種破壞模式           ← 你選
Phase 4 → 不做清單確認             ← 你定
Phase 5 → 桌面演練                 ← 你跑
Phase 6 → 最終判斷                 ← 你下
Phase 7 → 按發送                   ← 你按

第三件：紅隊 Prompt 是用來打你自己臉的。

Phase 4、5、7 都有紅隊 Prompt。用法：

  1. 跑完主 Prompt。拿到產出。
  2. 把產出貼進紅隊 Prompt。
  3. AI 會找你的漏洞。
  4. 不要辯解。調整。

如果紅隊讓你覺得整個策略都有問題——恭喜。你在花錢之前發現了。
