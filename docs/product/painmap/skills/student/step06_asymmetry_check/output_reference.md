# Step 06 不對稱動機檢驗 — 學生產出範本

> **版本**：學生演練版（產出範本）
> **用途**：對照自己的產出，確認格式和深度是否達標
> **說明**：以下是延續 Step 05 示範場景（LINE 常見問答自動回覆服務）的完整產出範例

---

## 產出範本

```markdown
# 不對稱動機檢驗 — 2026-04-09

## 我的產品

- **服務**：幫 Shopee 小賣家設定 LINE 常見問答自動回覆
- **定價**：NT$800（一次性）
- **目標族群**：月營收 3-10 萬的個人賣家，1 人獨立作業

---

## 不對稱動機判斷

| 問題 | AI 分析摘要 | 我的判斷 | 結論 |
|------|------------|---------|------|
| Q1 利潤保護 | 客服 SaaS 靠企業客戶賺錢，個人小賣家不在其利潤結構中 | 同意。我問過幾個用蝦皮賣東西的朋友，沒人聽過 Omnichat | ✅ |
| Q2 自我蠶食 | 月費 SaaS 若推 NT$800 一次性版，現有月費客戶會質疑定價 | 同意。而且 SaaS 的商業模式就是靠 MRR，一次性收費和他們的整個財務模型衝突 | ✅ |
| Q3 功能裁減 | 企業客戶依賴工單系統和多平台整合，競品砍不掉這些功能 | 同意，但我想到一個 AI 沒提的風險：競品可能推出「輕量子品牌」。不過這需要 6-12 個月，我的目標是 3 個月內驗證 | ✅（注意時間窗口） |
| Q4 銷售注意力 | 業務 quota 結構讓他們天然忽略 NT$800 的小客戶 | 完全同意，我自己在 Omnichat 填過試用表單（說我是個人賣家），等了 4 天沒有任何人回應 | ✅ |

## 結論：4 個 ✅ — 強不對稱動機

→ **行動**：進入 Phase 7，準備 GTM 一頁紙，發預售訊息

---

## 補充驗證計畫

- Q4 已驗證（試用表單測試）
- Q3 額外注意：若市場驗證超過 4 個月，重新評估競品是否有推出輕量版的動態
- Q1 + Q2：持續觀察競品定價頁是否新增入門方案（設定每月 check 一次的提醒）
```

---

## 品質對照清單

對照你自己的產出，確認以下每項都達到：

### 格式完整度

- [ ] 4 個 Q 都有 AI 分析摘要（不是原文貼上，是你自己消化後的一句話）
- [ ] 4 個 Q 都有你的人工判斷（不是「同意」兩個字，是說明你同意的原因或補充 AI 沒有的資訊）
- [ ] 每個 Q 都有明確的 ✅ 或 ⚠️
- [ ] 最後有總結表格 + 行動決定

### 判斷深度

- [ ] 至少 1 個 Q 的「我的判斷」欄位中，你補充了 AI 不可能知道的第一手觀察
- [ ] 如果有 ⚠️，你寫了「如何在 X 天內驗證」的具體方法
- [ ] 你沒有只寫「AI 說的很對」就結束——即使你完全同意，也要寫出「為什麼你同意」

### 行動清晰度

- [ ] 你的「結論」欄寫的是行動（前進 / 調整 + 怎麼調整 / 重選），不是「再想想」
- [ ] 如果是「前進」，你知道下一步是 Phase 7

## 常見錯誤範例（對照用）

### 錯誤範例 A：判斷欄太空洞

```
我的判斷：同意 AI 的分析。
結論：✅
```

問題：「同意」不是判斷，是複述。你需要說明「為什麼同意」或「AI 不知道什麼但你知道」。

更好的寫法：

```
我的判斷：同意。補充：我自己就是目標客群的朋友，
他們完全不知道 Omnichat 這個工具的存在，更不用說付月費了。
結論：✅
```

---

### 錯誤範例 B：把 ⚠️ 當成終點

```
Q3 結論：⚠️
→ 行動：不確定，再考慮看看
```

問題：⚠️ 需要一個「驗證計畫」或「調整方向」，不是停下來等待。

更好的寫法：

```
Q3 結論：⚠️
如何驗證：每個月查看競品的更新公告，
看他們是否推出輕量版或入門方案。
若 3 個月後沒有動靜，升級為 ✅。
→ 行動：前進 Phase 7，同時建立每月競品監測習慣
```

---

### 錯誤範例 C：競品分析太籠統

```
競品：某 SaaS 工具，比較貴，功能比較多
```

問題：「比較貴」「比較多」對 AI 沒有意義，它無法推論出不對稱動機。

更好的寫法：

```
競品：Omnichat，NT$2,000/月起，主要服務有客服團隊的電商品牌，
核心功能是多平台整合（LINE + FB + 官網）和客服工單系統
```

---

## 這份產出要交給誰看

不用「交給」任何人。這份判斷是給你自己的——它是你在發出預售訊息之前，
對「這個方向值得繼續」的最後一次理性確認。

如果 4 個 ✅ 都成立，你的任務是去 Phase 7，然後**真的發出那封訊息**。
