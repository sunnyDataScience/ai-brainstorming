# Step 04 策略圖設計 — 學生演練版 Prompt

> **版本**：學生演練版
> **對應技能**：`04_strategy_canvas.md` — Prompt 4A + Prompt 4B（紅隊）
> **預估時間**：30–45 分鐘
> **前一步需要**：Phase 3 的破壞機會卡片（破壞模式 + 目標族群 + 替代方案 + 切入角度）

---

## 背景知識（2 分鐘讀完，再開始填 prompt）

Strategy Canvas 是「讓你看見自己和競品有多不一樣」的工具。

核心邏輯：
- **橫軸**：客戶在意的功能維度（6–8 個）
- **縱軸**：每個維度的程度（高 / 中 / 低）
- **每條線**：競品的「功能曲線」

**你要做的事**：畫出你的曲線，讓它和競品的曲線**形狀不一樣**。

```
       高 │         ╱╲
 功能     │    ╱╲  ╱  ╲        ← 競品（什麼都做，很貴）
 程度     │   ╱  ╲╱    ╲
          │  ╱           ╲──   ← 你的產品（某些高、某些刻意不做）
       低 │╱
          └──┬──┬──┬──┬──┬──
            價格 易用 功能 速度 支援
```

**最重要的一句話**：
- 你的曲線和競品**形狀不同**（有些高、有些低）= 你在做不同的事 → 有差異化
- 你的曲線和競品**長一樣但全部更低** = 你只是更爛的版本 → 沒有差異化

## Prompt 4A：生成 Strategy Canvas

請把你在 Phase 3 得到的資訊填入 `{{ }}` 括號內，再完整複製貼入 AI 聊天工具。

```text
你是一位策略圖（Strategy Canvas）設計師。
我要針對一個破壞式機會畫出策略圖。

## 我的資訊

**問題**：{{Phase 2 一句話——你的目標族群遇到什麼重複性勞動或困境}}
**目標族群**：{{破壞機會卡片的目標族群，越具體越好}}
**破壞模式**：{{低端破壞 / 新市場破壞（選一個）}}
**切入角度**：{{破壞機會卡片的切入角度——你要用什麼方式切入市場}}

**現有替代方案 / 競品**：
1. {{競品或替代方案名稱}}：{{簡述做什麼、大約價位}}
2. {{競品或替代方案名稱}}：{{簡述}}
3. {{名稱 / 或「手動處理」/ 或「不處理」}}：{{簡述}}

---

請幫我：

### Step 1：定義功能維度

列出 6–8 個「客戶在選擇解法時真正在意的維度」。

規則：
- 用客戶語言，不是行業術語（「設定複雜度」→「多快能開始用」）
- 必須是可比較的（每個競品都能標高/中/低）
- 包含「價格」和「易用性」（幾乎所有場景都相關）
- 不要超過 8 個（太多 = 失焦）

### Step 2：標出競品位置

對每個替代方案，在每個維度標 高/中/低。
用表格呈現：

| 維度 | 定義 | 競品A | 競品B | 手動處理 |
|------|------|-------|-------|----------|
| 價格 | 每月成本 | 高 | 中 | 低（時間成本高） |
| ... | ... | ... | ... | ... |

### Step 3：建議「你的」差異化曲線

基於我的破壞模式和目標族群，建議我的曲線：
- 哪些維度應該拉到「高」（你的獨特優勢）
- 哪些維度應該壓到「低」（刻意不做）
- 每個建議都說明理由

**用 ASCII 圖呈現**：

```
維度：  價格  易用  功能數  速度  支援  整合  客製  安全
競品A：  ■■■   ■    ■■■■  ■■    ■■■  ■■■  ■■■  ■■■
競品B：  ■■    ■■   ■■■   ■■■   ■■   ■■   ■■   ■■
你的：   ■    ■■■■   ■    ■■■■  ■■    ─    ─    ■■
```

### Step 4：差異化摘要

用一段話總結：
「我用 [什麼優勢] 服務 [什麼人]，刻意不做 [什麼]，
因為 [為什麼不做比做更有力]。」

### 規則
- 不打分數（高/中/低 是相對位置，不是分數）
- 你的曲線必須跟競品「形狀不同」，不是「全部更低」
- 「刻意不做」的維度必須有理由
- 如果競品資訊不足，標注「需要你實際使用 / 看評論才能確認」
```

## Prompt 4B：挑戰你的策略圖（紅隊模式）

拿到 Prompt 4A 的回覆後，把完整輸出貼入下面的 `{{ }}` 括號，再複製貼入 AI。

```text
你是一位策略紅隊審查員。我剛畫了一張 Strategy Canvas，
請用以下 5 個問題挑戰我的策略。不要安慰我，直接說問題在哪。

我的 Strategy Canvas：
{{貼上 Prompt 4A 的完整 AI 回覆}}

---

### 挑戰問題

1. **假低端陷阱**：我砍掉的功能，我的目標使用者真的不需要嗎？
   還是他們嘴上說不需要，實際用到時會抱怨？
   → 怎麼驗證？

2. **「更便宜」不等於「破壞」**：我的曲線是不是只是「全部壓低」？
   如果是，我不是破壞者，我只是便宜貨。
   → 哪些維度我應該比競品更高？

3. **護城河風險**：競品如果也推出一個低價版，我怎麼辦？
   我有什麼是他們做了低價版也複製不了的？

4. **目標客群是否真的被忽略**：大公司真的不想做這群人的生意嗎？
   還是他們只是還沒做到？（如果「還沒做到」→ 你沒有時間優勢）

5. **「不做」的代價**：我「刻意不做」的功能中，有沒有哪一個
   在使用者旅程中是「一旦需要就完全卡住」的關鍵功能？

### 規則
- 不安慰我（「你做得很好」= 無用輸出）
- 每個挑戰都給出「怎麼驗證」的具體方法
- 如果我的策略有致命缺陷，直說
```

## 拿到 AI 回覆後要做的事

### 1. 調整你的策略圖

根據紅隊挑戰，回頭修改你的曲線。常見的調整方向：
- 把某個「刻意不做」改成「做基本版」（如果紅隊發現這個功能會卡住使用者）
- 把某個「高」降為「中」（不需要做到最好的維度）
- 發現一個被忽略的新維度，補上去

### 2. 寫出你的 Strategic No（不做清單）

把你的「不做清單」寫下來。這是整個策略最重要的部分：

```markdown
## 我的 Strategic No（不做清單）

1. 不做 ___，因為：___
2. 不做 ___，因為：___
3. 不做 ___，因為：___
4. 不做 ___，因為：___

原則：不做清單必須比「要做清單」長。
如果你的「不做」比「要做」少 → 你砍得不夠。
```

## 常見卡關與解法

### 卡關 A：「我不夠了解競品，維度標不出來」

你不需要深度研究。花 20 分鐘做快速競品體驗：
- 去競品官網看功能頁和定價頁
- 看 App Store / Google Play 的使用者評論（重點看差評）
- 問 AI：「幫我列出 {{競品名}} 的主要功能和使用者常見抱怨」

知道「他們大概做什麼、使用者大概抱怨什麼」就夠了。

### 卡關 B：「我的曲線跟競品長一樣，找不到差異」

兩個解法：
1. 加一個競品沒有的維度——例如「中文在地化」「一人公司友善度」「5 分鐘上手」
2. 確認你的目標族群和競品的客戶是不是其實不一樣的人。不一樣的人在意不一樣的維度，你的曲線自然就不同了

### 卡關 C：「紅隊問題讓我覺得整個策略都有問題」

這是好事。在你花錢做產品之前發現問題，比做了 3 個月才發現便宜太多了。

可能的處理方向：
- 回 Phase 3，選一個不同的切入角度
- 回 Phase 2，換一個更聚焦的問題主題
- 調整目標族群，縮小到你真的有辦法深入了解的一群人

每次重來都讓你更接近正確答案。
