# Step 06 不對稱動機檢驗 — 老師示範版 Prompt

> **版本**：老師示範版（課堂 live demo 專用）
> **對應技能**：`06_asymmetry_check.md` — Prompt 6A
> **Phase**：老師示範版 — Phase 6
>
> 老師說明：「我們已經設計好最小產品了——NT$800 幫蝦皮小賣家設定 LINE 常見問答自動回覆。
> 在發預售訊息之前，我還有最後一個問題要問：大公司會不會看到我的生意、直接做一個一樣的來搶我？
> 這個 prompt 幫我們用競爭動態分析師的角度，系統性地檢查這個問題。」

---

## 示範用 Prompt（直接複製貼入 AI 聊天工具）

```text
你是一位競爭動態分析師。我設計了一個最小破壞性產品，
在發出去之前，我想確認大公司不會輕易搶走我的客戶。

## 我的產品

**核心價值**：幫 Shopee 小賣家用簡單設定解決每天重複回 LINE 客服的痛，省下 1 小時
**目標族群**：月營收 3-10 萬的個人 Shopee 賣家，1 人獨立作業，客服全靠自己回 LINE
**定價**：NT$800（一次性）
**只做**：
1. 整理 10 個客戶最常問的問題 + 寫好回覆模板
2. 幫設定 LINE 官方帳號自動回覆關鍵字
3. 一週後檢查使用情況，調整 3 個回覆

**不做**：
- 不做訂單管理
- 不做多平台整合（FB、蝦皮訊息中心）
- 不做 AI 聊天機器人
- 不做數據報表或客服分析

## 我的主要競品 / 替代方案

1. 某客服 SaaS（如 ORCA、Omnichat）：NT$2,000/月起，主打企業客服團隊，多平台整合、工單系統、客服績效報表
2. LINE 官方帳號內建功能（自助設定）：免費，但介面複雜，小賣家不懂怎麼設定
3. 手動回 LINE：現行做法，每天花 1-2 小時，免費但消耗時間

---

請幫我做以下分析。每個問題都用「事實 → 結論 → 如何驗證」的格式回答。

### Q1：利潤保護

我的目標客群是競品的利潤來源嗎？

- 競品靠什麼客群賺最多錢？（企業客戶？付高價的專業用戶？）
- 我的目標客群在競品的營收中占多少比例？
- 如果競品也做一個我這樣的低價版，會不會搶到自己的高價客戶？

結論：競品「會 / 不會」為了保護我的客群而行動。
如何驗證：___

### Q2：自我蠶食

競品推出低價版會傷到自己嗎？

- 競品現在的定價結構是什麼？
- 如果競品推出 NT$800 的版本，他們的高端客戶會不會降級？
- 這個降級對競品的營收影響有多大？

結論：競品「會 / 不會」因為害怕自我蠶食而不推低價版。
如何驗證：___

### Q3：功能裁減

競品能不能砍功能來跟我競爭？

- 競品的企業客戶依賴哪些「我不做」的功能？
- 如果競品砍掉那些功能做一個簡單版，企業客戶會不會抗議？
- 競品的工程團隊願不願意維護兩個版本（完整版 + 簡單版）？

結論：競品「能 / 不能」輕易砍功能來跟我競爭。
如何驗證：___

### Q4：銷售注意力

競品的銷售團隊會重視我的客群嗎？

- 競品的業務一單做多大？（一單 NT$2 萬 vs 我的一單 NT$800）
- 業務會不會覺得「做這種小單不值得」？
- 競品有沒有 self-serve（自助註冊）的管道可以低成本獲取我的客群？

結論：競品的銷售團隊「會 / 不會」花力氣搶我的小客戶。
如何驗證：___

### 總結

| 問題 | 結論 | 風險等級 |
|------|------|---------|
| Q1 利潤保護 | 會/不會行動 | 安全 ✅ / 注意 ⚠️ |
| Q2 自我蠶食 | 會/不會推低價 | 安全 ✅ / 注意 ⚠️ |
| Q3 功能裁減 | 能/不能砍功能 | 安全 ✅ / 注意 ⚠️ |
| Q4 銷售注意力 | 會/不會搶 | 安全 ✅ / 注意 ⚠️ |

**整體評估**：
- 4 個 ✅ → 強不對稱動機，放心進入市場
- 3 個 ✅ + 1 個 ⚠️ → 可行，但需要速度——盡快拿到第一塊錢
- 2 個 ✅ + 2 個 ⚠️ → 有風險，考慮調整目標客群或定價
- ≤ 1 個 ✅ → 不是破壞式機會，回 Phase 3 重新選擇

### 規則
- 不打分數，不預測成功率
- 「安全 / 注意」不是分數，是事實判斷
- 每個結論都附「如何驗證」——因為這些都是假設
- 如果資料不足，直說「需要了解更多競品的營收結構 / 客戶構成」
```

## 老師備注

- 示範前強調：「我們不是要找『我比競品強』的理由，我們要找『競品不想來打我』的理由。這兩件事非常不同。」
- 貼入 prompt 後，邊等 AI 回覆邊說明每個 Q 在問什麼、以及為什麼這個問題重要
- 收到 AI 回覆後，先讓學生讀 30 秒，再引導討論：「AI 分析有沒有任何你覺得『它不可能知道』的東西？」
- 請學生對照 `output_sample.md` 討論老師的人工判斷，強調「AI 是第一層過濾，你的真實認知是第二層」
- 可以在課堂上故意問：「如果我的定價不是 NT$800 而是 NT$1,200，哪個 Q 的答案可能會改變？」
