## 7. URL 結構與路由

### 命名規範
- 小寫、連字符分隔: `/user-profile`
- 資源 ID: `/resources/:id`
- 巢狀資源: `/users/:userId/orders/:orderId`
- 查詢參數用於過濾/排序: `?status=active&sort=-created`

### 路由表

| 路由 | 元件 | 認證 | 載入策略 |
| :--- | :--- | :--- | :--- |
| `/` | HomePage | 否 | 預載 |
| `/login` | LoginPage | 否 | 懶載入 |
| `/dashboard` | DashboardPage | 是 | 懶載入 |
